## ABOUT THE COMPONENTS

### BUTTON
Can be seen on the main page .Clicking it will trigger the fetching of string.

### HISTOGRAM
Help us to graphically visualize the occurence of most used words.

### SPINNER
Appears when page is loading. When the button is clicked the fetching of string and find the top 20 frequently ised words requires some computation time so we can simply put spinner there.
